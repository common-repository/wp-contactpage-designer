<h3>  Features </h3>

<h3> Tool Box of Customizable HTML Fields and Controls</h3>
<ul>
<li>8 Customizable HTML controls available for contact forms
<li>Standard HTML controls: Label, E-mail, Submit Button, Paragraph, text area, Drop Down. 
<li>One click customization of field properties: size, color, font, weight, text alignment border color and more.
<li>Set fields to be mandatory or optional when users fill out contact forms
<li>Set Place Placeholder text in fields that disappears when users enters data in contact form.   
<li> Set email address for collected data to be sent.
<li> Receive an email instantly when someone completes your form.
<li> Customize text thank you message after form is submitted. 
</ul>

<h3>Store Contact form Templates</h3>
<ul>
<li> Create, edit, delete and save multiple contact form templates 
<li> Load individual contact form templates and Set desired template as default 
</ul>


<h3> Drag and Drop Controls Precisely </h3>
<ul>
<li> All fields are drag-able from Tool box to page canvas making contact forms easy to create. 
<li> Drag fields to precisely where you want them on canvas Or
<li> Set them precisely by using absolute positioning from the control’s setting. 
</ul>
